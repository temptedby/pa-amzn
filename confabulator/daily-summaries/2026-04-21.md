## Bugs squashed

- **ALERT_EMAIL append corrupted the Resend key**. `echo "ALERT_EMAIL=…" >> .env.local` tacked onto the end of a previous line that lacked a trailing newline, producing `RESEND_API_KEY=re_…rPxALERT_EMAIL=william@…` as a single line. Manifested as a 400 "API key is invalid" from Resend. Fixed by reading the correct key from Vercel (pre-corruption snapshot) and rewriting `.env.local` with proper line breaks. Lesson: use the server-action `updateEnvFile` helper for any append instead of raw shell `echo >>`.
- **TypeScript 5.7 `Uint8Array<ArrayBufferLike>` vs `ArrayBuffer`** — Web Crypto verify/sign rejected my default `Uint8Array` type. Fixed `fromBase64Url` to allocate an explicit `ArrayBuffer` and annotate the return as `Uint8Array<ArrayBuffer>`.
- **Recharts Tooltip formatter** — tightened typing in `SpendVsSalesChart.tsx` (later removed from the dashboard anyway).
- **Cron initially missed SP_API_REFRESH_TOKEN** because `source .env.local` failed on the `|` in the Amazon token (zsh interprets `Atzr|IwEBI…` as a pipe). Rewrote the Vercel env push to `grep | sed` per line so no shell interpretation runs on values.

## Strategy calls that mattered

- **Threshold model**: William asked whether Amazon should set thresholds automatically. Decided: keep manual numeric thresholds per SKU (user-set via inline input). Amazon's restock recommendations land in a separate read-only column. Daily cron will send an audit email if a future cron fails to refresh Amazon's recommendation data.
- **Shipment automation scope**: reversed yesterday's "build the full 8-step API chain" plan after Megan weighed in. Her time sink is computing quantities (which we can't automate — no historical demand signal without Ads data), not clicking through Send-to-Amazon. So: (1) deep-link out, (2) track status post-hoc, (3) skip the API chain entirely until shipment volume warrants it.
- **Domain verification**: went through it so branded emails from `alerts@phoneassured.com` work. Required 4 DNS records in GoDaddy; all propagated within minutes. Worth the 15 min.
